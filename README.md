# Job Import Tool - morich

求人票テキストをAIで解析し、PORTERS求人インポート用CSVを生成するツールです。

## 構成

```
job-import/
├── index.html      # フロントエンド（GitHub Pages）
├── gas/
│   └── proxy.gs    # GAS プロキシ（Anthropic API中継）
└── README.md
```

## セットアップ手順

### 1. GASプロジェクトの作成

1. [Google Apps Script](https://script.google.com/) を開く
2. 「新しいプロジェクト」を作成
3. `gas/proxy.gs` の内容をエディタに貼り付け
4. プロジェクト名を任意で設定（例：`morich-job-import-proxy`）

### 2. APIキーの設定

1. GASエディタ左メニューの「プロジェクトの設定」→「スクリプトプロパティ」を開く
2. 「プロパティを追加」をクリック
3. プロパティ名：`ANTHROPIC_API_KEY`、値：Anthropic APIキーを入力
4. 「スクリプトプロパティを保存」

### 3. ウェブアプリとしてデプロイ

1. GASエディタ右上「デプロイ」→「新しいデプロイ」
2. 種類：「ウェブアプリ」を選択
3. 設定：
   - **次のユーザーとして実行**: 自分（例：自分のGoogleアカウント）
   - **アクセスできるユーザー**: 全員
4. 「デプロイ」をクリック
5. 表示された **ウェブアプリURL** をコピー（`https://script.google.com/macros/s/xxxx/exec` 形式）

### 4. フロントエンドにURLを設定

1. GitHub Pages（または `index.html` をブラウザで開く）
2. 画面下部「⚙ GAS URLを設定する」をクリック
3. コピーしたウェブアプリURLを貼り付けて「保存」

これで利用可能になります。設定はブラウザの LocalStorage に保存されます。

---

## 使い方

1. 求人票テキストを貼り付け、またはファイルをアップロード
2. 企業名・ポジション・日付を入力
3. 「AI解析してCSVを生成」ボタンをクリック
4. プレビュー確認後、「CSVをダウンロード」

出力はBOM付きUTF-8 CSV（Excelで文字化けしない形式）。

## CSV仕様（28列）

| 列 | フィールド | 備考 |
|----|-----------|------|
| A | 企業名 | 入力値 or AI判定 |
| B | フェーズ | 固定: オープン |
| C | フェーズ日付 | 入力値 |
| D | ポジション | 入力値 or AI判定 |
| E | ざっくり職種 | AI判定（29職種から選択） |
| F | 労働局提出用区分 | 固定: 空白 |
| G | 職種 | 固定: 空白 |
| H | 役職 | AI判定 |
| I | 業務内容 | AI判定 |
| J | 応募資格(概要) | AI判定 |
| K | 応募資格(詳細) | AI判定 |
| L | 求める人物像 | AI判定 |
| M | 給与(詳細) | AI判定 |
| N | 勤務地 | AI判定（東京/神奈川千葉埼玉/大阪京都兵庫/その他） |
| O | 勤務地(詳細) | AI判定 |
| P | 雇用形態 | 固定: 正社員 |
| Q | 雇用形態(詳細) | 固定: 試用期間3ヶ月 |
| R | 選考プロセス | AI判定 |
| S | 年収下限 | AI判定（万円・数値のみ） |
| T | 年収上限 | AI判定（万円・数値のみ） |
| U | 年齢下限 | AI判定 |
| V | 年齢上限 | AI判定 |
| W | 非公開想定年齢 | AI判定 |
| X | マッチングメモ | AI判定 |
| Y | 受付年月日 | 入力値 |
| Z | 雇用期間 | 固定: 定めなし |
| AA | 有効期間開始 | 受付年月日と同値 |
| AB | データ登録日 | 実行日 |
